## anamorphic-desq.sh

```
Name
    anamorphic-desq.sh - Anamorphic Desqueeze Tool

SYNOPSIS
    anamorphic-desq.sh <-h|-v> <-f 1.55|1.33> <squeezed_file>

DESCRIPTION
    A script for quickly desqueezing anamorphic images

    <squeezed_file> is desqueezed using ffmpeg and stored in 
    <squeezed_file>_desq.png to avoid compression artifacts

OPTIONS
    -h
        The image should be desqueezed horizontally

    -v
        The image should be desqueezed vertically

    -f 1.55|1.33
        The squeeze factor
```
