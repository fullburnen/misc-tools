## dccli.sh

```
NAME
    dccli.sh

SYNOPSIS
    dccli.sh <action> compose_dir

DESCRIPTION
    A script used for managing docker compose containers.

    By default, it searches in ~/docker/ for compose_dir, which should container your compose file

OPTIONS
    action - pull
             down
             up - Runs detached
             down-up - Down followed by up
```
