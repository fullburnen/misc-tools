## ExifTool Sort

```
Name
    ExifTool Sort

SYNOPSIS
    exiftool_sort.sh [xavcs] unsorted_dir

DESCRIPTION
    A script used for copying and sorting a directory full of images and videos.

    The image and video files in unsorted_dir are copied into a
    YYYY/YYYY-MM-DD folder in the current directory.

OPTIONS
    xavcs - Updates Sony XAVCS format video files to have useful dates for
            sorting. Renames the files into a timestamp. Does not sort.
```
